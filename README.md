# Topicos especiais - analise textual

Repositorio com os notebooks e arquivos de dados utilizados na pratica de segmentacao, POS tagging, NER e analise de dados textuais.

## Estrutura

- Notebooks `.ipynb` na raiz do repositorio.
- `data/`: arquivos de dados gerados e utilizados nos notebooks.

## Arquivos de dados

- `data/documentos.csv`
- `data/dataset.zip`
- `data/datasetpos.zip`
- `data/datasetner.zip`
